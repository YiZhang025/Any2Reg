Optional demo data. Either copy real STONE/ACDC files into `stone/data`, `stone/features`, `acdc/data`, `acdc/feature`, or generate synthetic data with `python scripts/create_sample_data.py`. To keep feature files small, use `scripts/copy_real_data_to_sample.py` (writes only `logits_final`, 192×192) or `python scripts/shrink_features.py <full.npz> -o <out.npz>` on existing npz files.
